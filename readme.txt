== Description ==
At your Edit Post view, use shortcode as \[balloon-r\] or \[balloon-l].

Change balloon style at 'Tools'->'Speech Balloon Maker' screen.

You can also change style at shortcode parameter ex. \[baloon-r name='foo' style='border' img='https://foo.com/bar.jpg'\]

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/speech-balloon-maker` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Tools'->'Speech Balloon Maker' screen to configure the plugin
